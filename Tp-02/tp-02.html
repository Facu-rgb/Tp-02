1.1: que es una url?

La "Uniform Resource Locator"(URL o Localizadora Uniforme de Recursos en Español) es una cadena de texto que especifica dónde se puede encontrar un recurso (como una página web, una imagen o un video) en Internet.

1.2: que es una url absoluta?

Una URL absoluta es una dirección web completa que incluye el protocolo, el nombre de dominio y la ruta completa del archivo o recurso

1.3: quees una url relativa?

Las URL relativas son estructuras de URL que contienen la ruta de ubicación, omitiendo el nombre del sitio web

1.4: cual es la diferencia entre una url relativa y una absoulta?

Una dirección URL absoluta contiene toda la información necesaria para localizar un recurso. Una dirección URL relativa busca un recurso mediante una dirección URL absoluta como punto de partida

1.6: que es un directorio? que relacion tiene con una carpeta?

A diferencia de una carpeta, que sólo puede almacenar archivos, un directorio puede almacenar archivos, subdirectorios y otros directorios . En su nivel más básico, un directorio es un sistema organizativo que permite a los usuarios almacenar archivos de datos de una manera que los haga fáciles de encontrar

1.7: que es un directorio raiz y con que simbolo se representa?

Un directorio raíz es el nivel superior directorio or carpeta en un sistema de archivos jerárquico. Se lo representa con una barra inclinada (/)

1.10: cual es el proposito de la etiqueta anchor?

el elemento anchor crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL

1.11: que atributo se usa en la tiqueta anchor para especificar la url a la que se debe enlazar?

el aributo que se usa para especificar la url de la etiqueta anchor es href

1.12: que sucede si el atributo "href" en la etiqueta anchor esta vacio?

cuando el atributo href no esta presente en la etiqueta anchor esta ultima no puede anclar un hipervinculo
